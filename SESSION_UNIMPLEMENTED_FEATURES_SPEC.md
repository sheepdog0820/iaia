# セッション機能 未実装・部分実装仕様

最終確認日: 2026-06-12

この文書は、現行実装を確認したうえで残っているセッション関連の候補だけを整理します。実装済み機能の全体像は `docs/CURRENT_WEBAPP_FEATURES.md`、課題の優先順位は `ISSUES.md` を正本とします。

## 実装済み

以下は旧版の本書では未実装扱いでしたが、現在は実装済みです。

- セッションノート・ログ
- セッション画像と複数YouTubeリンク
- 定期セッション、複数日程、参加可能日・日程調整投票
- セッション分析ダッシュボード
- 経験値・報酬配布
- セッションテンプレート
  - 基本情報、グループ、シナリオ
  - 画像、秘匿ハンドアウト雛形
  - カレンダー入力からの即時登録・詳細編集
  - セッション作成時の画像・ハンドアウト複製

## 部分実装

### キャラクター統合

- 参加者へのキャラクターシート紐付けとHP/MP/SAN参照は実装済み
- セッション開始時スナップショット、状態差分、継続的なステータス効果管理は未実装

### セッション素材

- セッション画像、YouTubeリンク、ハンドアウト添付は実装済み
- マップ、BGM、文書を横断管理する汎用素材ライブラリとカテゴリ分類は未実装

### 外部連携

- iCalファイルエクスポートは実装済み
- 購読トークン付きICSフィード、Google Calendar双方向同期、Discord Webhook、VTT連携は未実装
