# クトゥルフ神話TRPG 7版 キャラクターシート仕様書

**ステータス**: 正式サポート  
**最終更新**: 2026年6月17日

## 概要

タブレノの7版キャラクターシートは、クトゥルフ神話TRPG 7版のパーセンテージ能力値をそのまま保存する。6版との互換性を保つため、共通の `CharacterSheet` に基本情報、能力値、HP/MP/SAN、技能、装備、画像を保持し、7版固有の派生値は計算済みレスポンスとして提供する。

## 対応範囲

- 作成画面: `/accounts/character/create/7th/`
- 作成API: `/api/accounts/character-sheets/create_7th_edition/`
- 版別一覧API: `/api/accounts/character-sheets/by_edition/?edition=7th`
- 詳細/編集/一覧画面での7版表示
- セッション作成・テンプレート・シナリオ連携での7版選択
- CCFOLIA JSONインポート時の7版判定

## 能力値と派生値

### 能力値

- STR, CON, POW, DEX, APP, SIZ, INT, EDU は7版のパーセンテージ値として扱う。
- 画面の標準ダイス設定は `3D6×5`、SIZ/INT/EDU は `2D6+6×5`。
- DB上は既存互換のため `1..999` を許容する。UIと標準作成フローでは15..90相当を基本とする。

### 派生値

- 最大HP: `(CON + SIZ) // 10`
- 最大MP: `POW // 5`
- 初期SAN: `POW`
- 最大SAN: `99 - クトゥルフ神話技能`
- 幸運: `POW`
- 回避初期値: `DEX // 2`
- 母国語初期値: `EDU`

### ダメージボーナスとビルド

`STR + SIZ` を基準に以下を返す。

| 合計 | ダメージボーナス | ビルド |
| --- | --- | --- |
| 0-64 | -2 | -2 |
| 65-84 | -1 | -1 |
| 85-124 | +0 | 0 |
| 125-164 | +1D4 | 1 |
| 165-204 | +1D6 | 2 |
| 205-284 | +2D6 | 3 |
| 285-364 | +3D6 | 4 |
| 365-444 | +4D6 | 5 |
| 445+ | +5D6 | 6 |

### 移動率

- STRとDEXがどちらもSIZ未満: 7
- STRとDEXがどちらもSIZ超過: 9
- それ以外: 8
- 年齢40代から10歳ごとに1ずつ減少し、最低値は1。

## 技能ポイント

- 職業技能ポイント: 標準は `EDU × 4`
- 職業選択により `EDU×2 + APP/DEX/POW/STR/CON/SIZ×2` もUIで選択可能。
- 趣味技能ポイント: `INT × 2`
- 技能の現在値は `基本値 + 職業P + 趣味P + その他P` を基本とする。

## APIレスポンス

`CharacterSheetSerializer` は7版キャラクターに `character_7th` を含める。

- `damage_bonus`
- `build`
- `move_rate`
- `dodge`
- `current_luck`
- `max_luck`
- `occupation_points`
- `interest_points`

6版互換キーとして `character_6th` も返す。7版の場合は `character_6th: null`。

## 品質基準

- 7版作成API、派生値、技能ポイント、装備保存、一覧/詳細表示をテスト対象にする。
- 6版の既存挙動を変更しない。
- 7版関連の「開発保留」文言はユーザー向け画面、ガイドライン、仕様書から除去する。
